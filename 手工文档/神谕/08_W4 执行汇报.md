## §0 一句话结论

**G2/G2R 未立**：学习臂在冻结双门（exact ≥0.640 ∧ 保费回收 ≥47%）内始终无达标点，且"过门 π 带 = 空"跨**两个世界锚**（v2 合成锚 + HumDial 实测重锚）复现，与 LOO 上限 43.9% < 47%、gain-free 结构三方互证——**门在该特征类 + 策略类的可达域之上**。学习组件按预注册收口（rung 4 永久收枪、v3 无 C2），转论文分析章。但 W4 产出三类硬资产：① **部署前沿与两个支配点**（含对 fixed 的无损弱支配，跨世界锚稳健）；② **三轴失败解剖**（排序/世界/校准，逐轴带收据的定量归因）；③ **探针负结果**（结构特征之外，文本/韵律/Omni 内部读数对停时无增量信号——特征收缩的真实数据背书）。

## §一 阶梯 rungs 2–3：零训练自适应臂（7/09 晚，近零 GPU：决策缓存 214–217 hits/臂）

四臂全败双门（目标区 = 回收 ≥47% 且 exact ≥−1pt 对 fixed 0.640）：

| 臂 | exact | 回收 | 判 |
|---|---|---|---|
| κ 规则 v0 | 0.550 | 62.5% | exact 崩 |
| κ 规则 safe | 0.630 | 30% | 回收不足 |
| κ 规则 rev | 0.650 | 保费↑133.1（负回收） | kill 判据未触发 = κ 对齐有效但方向反 |
| prompted-finality | 0.610 | **84%** | exact −3pt |

**机制四钉**（probe 收据 `w4_ladder_design` §7）：① 早提交**关闭修订动作类**（fixed 靠 rescued_patch 赢的夹，短窗臂 patches=0）；② **afterthought 修订对尾韵律不可见**（rollback EoU 73% 被标 final）——"韵律测话语完成，不测意图稳定"；③ 修订发生率是位置/话语结构变量非 κ 变量；④ 间隙落点与 D3 预算阈值咬合。**零训练前沿 = fixed(0,0) / safe(30%,−2) / pf(84%,−4)** → rung 4 停时头目标区由此反推固化。

## §二 rung 4 三代（v0 → v1 → v2，各自预注册；`w4_ladder_design` §8–12）

### v0（死因：代价函数/记账层）

- 训练：N=8000 合成（`config_hash=d48204c6582b`），19,531 ops / 205,242 hazard 样本 / 3.00% 正例；**val AUC 0.789**、校准贴合；合成集选 `c_w=0.08`。
- FDB：决策缓存 215/3、finality 217/0；exact **0.580** / state 0.620 / 保费 6.2s / 预注册回收 **97.4%**（三臂共同支撑 98 夹口径 94.4%，`housing_25` blocking done=None，不改判决）/ Δexact **−7pt**（0 gain / 7 loss）→ 回收门过、exact 门差 6pt，**AND 失败**。
- 尸检：等待逐 op 均匀收费 + miss 定价 3s ↔ 真实记账（保费只在末段窗/尾溢出产生、miss≈二元死刑）错配。

### v1（死因：世界先验/分布层）

- 生成器显式时间轴 + rescue 后负样本 + 训练器时间轴真实回放代价 + KILL_PEN=50（G2' 汇率导出）。`config_hash=497567920fc4`，19,402 ops / 6,310 rescue states / 382,869 hazard；**val AUC 0.859**、合成选点 `c_w=0.002`（训练 final_eou 窗 3.820s 已预告保守失守）。
- FDB：缓存 210/8、217/0；exact **0.640 精确踩门** / state 0.640，但保费 **318.6s**、主回收 **−193.1%**（strict −190.6%, n=98）→ 回收门崩，**AND 失败**。v0 七夹收复 6/7（`ecommerce_25#1` 仍缺）、gain `ecommerce_15#0`、新 loss `ecommerce_23#0`，净 −1pt。
- 尸检：合成 32.5% 修订先验 → 风险读数系统性偏高（诚实预测的"真赌点失守"分支兑现）。**不得用 FDB 反调**（防火墙），修法只许非评测数据。

### §11 因果上限诊断（`w4_ceiling_diag.py`，零 GPU）

FDB 特征上信号存在：**LOO AUC 0.753**（载荷 = slots_missing / f_unfinished / 短 utt_dur / 早 eou_idx）；屏障修正（有效 gap = gap−1.0；patch 类密集 1.64s = hold+infer）后 **W=1.5 覆盖全部 13 个承重 op**；LOO 前沿 lost=1 → **43.9%**（目标区角 47 在其上）；**瓶颈 = 排序迁移**（v0/v1 模型迁移 AUC 0.679/0.644 → lost=1 仅 18%/8%）。

### v2（最后一枪；死因：迁移层——排序已闭合、风险标度未闭合）

- 五件套：域随机化（RANGES 逐 dialogue 采样 + GAP_FLOOR=1.64 结构勘正）/ FEATS_V2 收缩至 7 维 / **二段式策略**（risk=∫λ̂@[0,2.5)≥θ → 护 1.5s 否则立即提交；protect-all ≡ fixed ⇒ v1 式 −193% 不可表达）/ 屏障宽限入训练回放 / 低带（rev_intensity 下三分位）联合选型。
- 训练：gen `config_hash=b62a069cd900`（8,000 dlg / 19,288 ops / 池化修订 24.6% / rev_prior p10/50/90 = .077/.198/.474）；标签 394,757 hazard / 正例 1.20%；**LR val AUC 0.749 / MLP 0.896**——MLP 判别力更高但低带经济学塌 protect-all，**容量涨判别不涨部署经济学**（升容量判据因此不触发）；联合选型取 **LR θ=0.03**（θ\* 第三次精确命中 1.5/50 盈亏锚）。
- FDB：决策缓存 217/1、finality 217/0 ⇒ 翻转纯窗口政策效应；exact **0.630**（差 1 夹）✗ / done50 3.455=fixed 逐位 / 保费 82.6s / 回收 **24.8%**（strict 24.7%, n=98）✗ → **AND 双败，rung 4 按预注册永久收枪**。0 gain / 2 loss（eco01#1、eco25#1）= v0 七夹严格子集；windows 严格两点 {0×27, 1.5×142}（protect 84.0%，κ-平坦）。
- 机制：θ=.03 经济学要求 ~2/3 op 读出风险 <3%，迁移后 84% 被读 ≥3% ——**排序 × 风险标度双重迁移误差**；lost=2 处 LOO 上限 52% vs 实测 24.8%，残差 ≈ §11 预言量级。
- **transfer 回填（§12.8）**：三代迁移 AUC **0.679 → 0.644 → 0.752009**，与 LOO **0.753157** 差 0.001148——**排序数据轴已实质榨干**；但部署点 protect@θ=85.7%，容量判据(i)（lost≤1 前沿差 ≤5pt）实测 38.4 vs 43.9 = 5.5pt，**以 0.5pt 严格未立**；剩余失败收缩为**风险绝对标度/校准迁移**。

## §三 W4-V3（G2R：真实数据轴；`docs/w4v3_design.md` 全文）

**前提变更**：用户取得 HumDial HD-Track2 完整训练许可（9,988 训练样本/106.9h/中英双语）；两段式预注册（结构先冻结、数值 Phase-1 回填后冻结）；三臂 C（校准-only）/ F（特征升级，探针门控）/ P（三档窗，F 门控）；合规红线 = 产物零转写原文。

### Phase-1（普查/探针/Omni 读数；§9）

- strict 对表 train/dev **9,988/1,800 全过**；VAD 检出 `break_mid` **748**（kill <300 不触发）；实测 max-pause p50 zh **0.900s** / en **1.092s**（显著长于 v2 合成 fast 带 [0.223,0.607]——复合 kill 的"分布无差"支不成立）。
- **主文本探针 FAIL**：1,143 对 PAIRED-CONTINUATION，结构族 S AUC **0.9841**（近饱和），S+T1+T2 **0.9662**，ΔAUC **−0.0179**/p≈1；**Omni X 族独立门 FAIL**：paired common support 745 对，S 0.9897 → S+X 0.9916，ΔAUC **+0.0018**（<0.05 门）。⇒ **Arm F 终局不启用、Arm P 不进入，仅余 Arm C**。"结构之外文本/韵律无正增量" = FEATS_V2 收缩的真实数据背书。
- 附带仪器发现：Omni finality 在真实停顿上 `break_mid→unfinished` **85.3%**、`utt_end→final` **70.0%**；hesitant 发射率 0（FDB 上 38/217——判官行为随域移动）；amend v0 塌缩（1890/1895 判 continue）= 无效仪器。

### Arm C0（先验移位，零训练；§10–§11）

- 定版：v2 权重逐位不动，θ 重映射 `θ_eff(π)=σ(logit(.03)−logit(π·q_H)+logit(m_train))`；冻结常量 **q_H=731/748**、**m_train=3307/19288**；π 网格 8 点 {.02–.30}、π\*=.10；预测 P1–P5 跑前锁定。
- 8 点全量（每臂 100/100、217 决策、finality 217/0）：θ .243→.0152，protect .066→.988，exact **.570→.650**，回收 **143.3%→6.5%**；π\* 行 exact .600 / 回收 101.6%（0 gain/5 loss）。
- 判读：**P1 空带兑现**（全网格无 AND 点）；P2 单调兑现；P3 π\* 点 3/4 MISS（根因 = risk 读数质量悬崖：θ∈(.0152,.0558] 内 protect 98.8%→40.0%）；**P4 落点 1/3 带内 FAIL → C1 按冻结规则触发**。网格夹逼给出持久夹闭式风险带：eco25#1 ∈[.0152,.0257)、eco01#1 ∈[.0257,.030)（护 eco01 成本 ≈46 回收点）。

### Arm C1（世界重锚重训；§11.4–§12）

- C1 世界（经验停顿池重采样 + 发射行实测混淆）：`config_hash=cca0eb049af8`，19,312 ops / rescuable **43.08%**（vs v2 69.7% = 世界锚差异实体）⇒ m_train=**.10429**；LR val AUC **.74847**（MLP stdout .902，冻结规范仍取 LR）。
- 8 点全量（决策缓存 1740H/1M、finality 1736H/0M）：exact .570→.650，回收 **137.1%→−4.0%**；π\* 行 exact **.620** / 回收 **74.1%**；全臂 0 gain、loss 严格嵌套；π=.20/.30 **零 loss**。
- **终局（§12）**：**P5 零假设被决定性拒绝**（8/8 点回收带外，Δ = −6.2/−4.3/−2.1/−4.9/−27.5/−54.9/−8.5/−10.5）⇒ **世界锚承重**；分化两成分定量拆开（公式成分：真实停顿更长 ⇒ m_train .171→.104 ⇒ θ_eff 全网格下移；风险场成分：同 protect .91 下 C0 14.7% vs C1 5.9% 同 lost=1，但 0-loss 点提前、持久夹排名上升）。**AND 门在 C1 同样空带 ⇒ P1 空带跨双世界锚复现**，与 LOO 43.9<47、gain-free 三方互证——**G2R 未立，v3 无条件收口，无 C2**。

## §四 正资产清单（论文素材，全部带收据）

1. **部署前沿（双臂并集）**：0 夹 6.5% / −1 夹 14.7% / −2 夹 **60.8%** / −3 夹 **74.1%** / −4 夹 84%(pf) / −5 夹 124.1%。全部为真实部署链路（无 oracle、无测试集接触）。
2. **两个支配收据**：C0@π=.30 **无损弱支配 fixed**（exact 相等、0 loss、保费 102.7<109.6；跨世界锚稳健——C1@π=.20 6.2%@0-loss 同性质）；C0@π=.15 (60.8%,−2) **严格支配零训练 safe** (30%,−2)。π 是用户可感的部署旋钮（闭式校准，零重训）。
3. **三轴失败解剖**：排序轴（迁移 AUC 0.679→0.644→0.752 追平 LOO 0.753）/ 世界轴（P5 八点拒绝 + m_train 位移的机械预测全兑现）/ 校准轴（先验移位公式成分全中、绝对落点被世界模型误差支配；关闭它所需的联合 (状态,修订时序) 域内数据恰为防火墙+域差所拒绝之物）。
4. **探针负结果**：真实数据上结构特征近饱和（S 0.984），文本连续性/韵律代理/Omni 内部读数均无增量——为"7 维结构特征 + 二段式策略"的设计选择提供数据背书；标点族 P=标注员泄漏（0.9979）的甄别方法学本身可写。
5. **仪器与方法学**：两段式预注册、闭式风险带夹逼、θ\* 三次精确命中盈亏锚、合规层（`assert_no_text`）、域随机化生成器（默认路径逐字节保持 `b62a069cd900`）。

## §五 诚实清单（勘误与边界）

1. v0 回收主口径 97.4% 与三臂共同支撑口径 94.4% 并存（housing_25 blocking done=None），判决不受影响。
2. v1 保费 318.6s 属训练窗畸大（final_eou 3.820s）直接后果，跑前已预告。
3. C1 π=.15/.30 完成锚小偏差（premium p50 1.575/1.576；π.30 protect-all = fixed+4.3s）——outcome 层与 fixed 完全一致，候选 = workers-12 tool_wall 抖动（已知非权威 regime）或 parse-repair 单夹微移，量级 ≪ 判据承重差。
4. MLP 全程判别力 ≥ LR 但部署经济学更差（低带塌 protect-all）——"容量涨判别不涨经济学"是结论不是省略。
5. sim-to-real 两风险（模拟 finality→真 Omni、脚本状态→Omni 决策）在 v0/v1 轮已声明、本程序内不可拆分；P5 之后其中"世界层"成分已被 C1 定量坐实。
6. P5 预测本身（C1≈C0）未中，如实记账：预注册的价值恰在于让这个"未中"成为可解释的测量。

## §六 裁定：FDB 刷分暂缓（用户 7/15，据 chat 讨论）

**裁定**：暂缓一切以抬高 FDB 绝对分为目标的工作（含蓝图#8 ASR 重听试点的执行、实体/ASR 类修复批次、judge 轨口径再优化）。理由：

1. **统计意义薄**：n=100，±9pt 量级二项噪声；门与差距在 1–2 夹刀口上。
2. **剩余失败主体两臂共通**（ASR/实体逐字、决策模型限度、hou11 类 2/100 全系统不可赢、pop(0) 对齐怪癖）——修它们抬绝对电平、不动平价结论，且逼近测试集过拟合红线。
3. **论文主张是平价 + 延迟 + 修订安全**，不是 FDB 榜首；v3.1 已按预注册封盘（两轮纪律用尽，两臂同升 +9/+10pt 的合法优化已做完）。
4. 已根因化的合法杠杆（#8 试点 TRIGGERED、实体规范化、judge 轨定位）**保留台账、转入多 bench 评测阶段再议**——在更大、修订密集的评测面上它们才有统计意义。
